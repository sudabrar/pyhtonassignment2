## ✨ Features

### Core Features
- **Course Management:** Create and manage courses with capacity limits
- **Student Registration:** Register regular and premium students
- **Enrollment System:** Enroll/drop students from courses
- **Capacity Management:** Automatic capacity tracking and full course detection
- **Data Persistence:** Save and load data using JSON files
- **Error Handling:** Comprehensive input validation and error messages

### Bonus Features
- **Inheritance:** PremiumStudent class with capacity bypass privileges
- **File Handling:** Save/load system state to JSON files
- **Interactive Menu:** User-friendly command-line interface
